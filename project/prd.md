# 프로젝트 이름 : 지하철 위치 및 아파트 실거래가 관계 분석

## 목표

지하철역 개통 시점을 기준으로 전후 5년간의 아파트 실거래가를 분석하여 지하철역 개통과 아파트 실거래가의 관계를 분석

## 데이터

- 지하철역 개통 정보
- 아파트 실거래가 정보

## 분석
